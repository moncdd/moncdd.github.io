9.1.1. Instalación de Symfony en XAMPP

Consideraciones previas

Si accedemos a la documentación oficial de Symfony (https://symfony.com/doc/current/index.html), para trabajar con este framework es necesario tener instalado una serie de herramientas:

  • PHP 8.2 o superior y las siguientes extensiones: Ctype, iconv, PCRE, Session, SimpleXML, and Tokenizer;
  • Composer: es una herramienta para la gestión de dependencias en PHP. Permite declarar las bibliotecas de las que depende el proyecto y las gestionará (instalar/actualizar) por nosotros.
  • Git: es un sistema de control de versiones distribuidos diseñado para manejar todo, desde pequeño hasta proyectos muy grandes con velocidad y eficiencia.
  • Symfony CLI (opcional pero recomendado): proporciona todas las herramientas que se necesitan para desarrollar y ejecutar la aplicación Symfony localmente.

Entorno Windows

A continuación se va a describir cómo instalar Symfony en XAMPP en un entorno windows:

  • Instalamos composer desde su página oficial (https://getcomposer.org/donwload/)

  • Instalamos git desde su página oficial (https://git-scm.com/downloads)

  • Configuramos el usuario y el correo electrónico usado en git

  • Instalamos Symfony CLI, para ello tenemos que acceder a Powershell para modificar la directiva que nos permitirá instalar el entorno

  • e instalamos Symfony CLI desde el intérprete de comandos

  • Comprobamos la instalación

  • y creamos un proyecto de prueba para ver si funciona

Entorno Linux

A continuación se va a describir cómo instalar Symfony en XAMPP en un entorno linux:

  • Instalamos composer

  • Instalamos git

  • Instalamos Symfony CLI

  • Comprobamos que el entorno está configurado
