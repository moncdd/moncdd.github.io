5.1. Ejecutar código en XAMPP

Generalmente, un archivo PHP contiene etiquetas HTML y algún código de secuencia de comandos PHP. Es muy fácil crear un ejemplo PHP simple. Para hacerlo, cree un archivo y escriba etiquetas HTML + código PHP y guarde este archivo con la extensión .php.

Todo el código PHP va entre la etiqueta <?php y ?>. La sintaxis de la etiqueta PHP es de la siguiente manera:

<?php   
           //your code here  
?>  

Un ejemplo simple de PHP en el que escribimos texto usando el comando echo de PHP es el siguiente:

<!DOCTYPE>  
<html>  
   <body>  
      <?php echo "<h2>Hello First PHP</h2>"; ?>  
   </body>  
</html>  

Los programas se guardan con la extensión .php, es decir, nombre de archivo.php dentro de la carpeta htdocs de XAMPP, que corresponde a la carpeta por defecto del servidor Apache en Windows. Por ejemplo, si estoy usando Windows y mi servidor XAMPP está instalado en la unidad C, la ruta del directorio htdocs será "C:\xampp\htdocs".  El programa PHP se ejecuta en un navegador web como Chrome, Firefox, etc, y escribimos en la URL la dirección del servidor local http://localhost
