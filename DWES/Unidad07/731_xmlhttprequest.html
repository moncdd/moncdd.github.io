7.3.1. XMLHttpRequest

Un objeto de XMLHttpRequest se utiliza para la comunicación asincrónica entre cliente y servidor. Éste realiza las siguientes operaciones:

  1. Envía los datos del cliente en segundo plano
  2. Recibe los datos del servidor
  3. Actualiza la página web sin recargarla.

Las propiedades comunes del objeto XMLHttpRequest son las siguientes:

Propiedad Descripción
onReadyStateChange Se llama cada vez que cambia el atributo listo. No debe utilizarse con peticiones sincrónicos.
readyState

Representa el estado de la solicitud. Va de 0 a 4.
0 UNOPENED: open() no se llama.
1 OPENED: se llama a open() pero send() no se llama.
2 HEADERS-RECEIVED: se llama a send() y se han recibido las cabeceras de la respuesta
3 LOADING: se recibe el cuerpo de la respuesta como responseText que contiene los datos.
4 DONE: la operación se ha completado por completo y se ha recibido la respuesta.
reponseText Devuelve la respuesta como texto.
responseXML Devuelve la respuesta como XML

Los métodos más importantes del objeto XMLHttpRequest son los siguientes:

Método Descripción
void open(method, URL) Abre la solicitud especificando el método de la petición y url.
void open(method, URL, async)

Igual que arriba, pero especifica si la petición es síncrona o asíncrona. En el primer caso, la ejecución se detiene hasta que se recibe la respuesta. En el segundo, la ejecución continúa y, cuando se recibe una respuesta, se procesa.

  • async = true. La petición se realiza de manera asíncrona
  • async = false. La petición se realiza de manera síncrona
void open(method, URL, async, username, password) Lo mismo que arriba, pero especificando el nombre de usuario y la contraseña.
void send() Envía la solicitud get.
void send(string) Envia la solicitud post incluyendo las variables enviadas como parámetros.
setRequestHeader(header,value) Añade cabeceras de petición.

Como se puede ver, el objeto XMLHttpRequest juega un papel importante:

  1. El usuario envía una solicitud de la interfaz de usuario y una llamada javascript va al objeto XMLHttpRequest.
  2. HTTP Request es enviado al servidor por el objeto XMLHttpRequest.
  3. El servidor interactúa con la base de datos usando un lenguaje de programación del lado del servidor.
  4. Los datos se recuperan.
  5. El servidor envía los datos en formato XML o JSON como respuesta a la función XMLHttpRequest.
  6. Los datos HTML y CSS se muestran en el navegador.
