1.7. Asignación de variables por referencia

En principio, la asignación de variables se realiza mediante copia, esto es, se crea una nueva variable y se le asigna el valor de la variable origen.

$a = $b; //se crea una variable a y se le asigna el valor que tenga b

También es posible definir una referencia a una variable utilizando el operador &:

$var2 = &$var1; //en este caso, var2 no es una nueva variable, si no que apunta a la misma dirección de memoria que var1
